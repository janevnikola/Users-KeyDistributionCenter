Комуникација помеѓу корисници и сервер KeyDistributionCenter
Имплементација:
Корисник Alice сака да комуницира со Bob,
Alice испраќа request до KDC кој го враќа клучот на Bob 
KDC ги содржи сите клучеви на корисниците
